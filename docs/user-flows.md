# Parcours utilisateurs

## Parcours 1 — Résident visiteur froid
1. Arrive sur la landing page
2. Comprend la promesse en moins de 10 secondes
3. Lance le diagnostic gratuit
4. Répond au questionnaire
5. Découvre sa restitution
6. Laisse son email pour recevoir ses recommandations détaillées
7. Reçoit une séquence email segmentée

## Parcours 2 — Résident en recherche d'aide rapide
1. Arrive via SEO sur une ressource / FAQ
2. Consulte une fiche pratique
3. Voit une invitation douce vers le diagnostic
4. Passe le diagnostic
5. Est orienté vers les ressources / offre adéquates

## Parcours 3 — Résident déjà qualifié / intéressé premium
1. Passe le diagnostic
2. Reçoit un profil à forte tension ou besoin complexe
3. Se voit proposer un appel / accompagnement / offre premium
4. Laisse ses coordonnées
5. Entre dans un suivi CRM ciblé

## Parcours 4 — Utilisateur ressource
1. Entre dans la bibliothèque
2. Filtre par besoin
3. Consulte 2-3 ressources
4. Est orienté vers diagnostic ou copilote

## Parcours 5 — Utilisateur copilote
1. Ouvre l'agent conversationnel
2. Pose une question concrète
3. Reçoit une réponse spécialisée + ressources liées
4. Est invité à approfondir via diagnostic ou offre premium
